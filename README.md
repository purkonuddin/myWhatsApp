## How To Install

1. Clone this repository
   ```
   $ git clone https://github.com/purkonuddin/myWhatsApp.git
   ```
2. Install all depedencies on the package.json
   ```
   $ npm install
   ```
3. Run
   ```
   $ npx react-native run-android
   ```
